# Seed Store – Plant Medicines

أدوات جاهزة لإضافة أدوية علاج أمراض النباتات لمتجر التطبيق (الـ Store) على السيرفر.

## الملفات
- `products.json` — 12 منتج في 4 فئات: مبيدات فطرية، مبيدات حشرية، مبيدات بكتيرية، أسمدة ومنشطات. (عدّل الأسعار/الأوصاف زي ما تحب)
- `upload_medicines.py` — سكريبت يعمل لوجين، يولّد صورة لكل منتج، ويرفعه على endpoint الـ `add-product`.

## المتطلبات
- Python 3 + مكتبة requests:
  ```
  pip install requests
  ```
- (اختياري) Pillow عشان الصور تطلع عليها اسم المنتج: `pip install Pillow`. من غيرها بيتولّد صورة بلون الفئة.

## الخطوات
1. شغّل السيرفر بتاعك و ngrok، وخُد الرابط الجديد (لازم ينتهي بـ `/`).
2. حدّث نفس الرابط في `lib/core/networking/end_boint.dart` (سطر `baseUrl`) عشان التطبيق يكلّم نفس السيرفر.
3. شغّل السكريبت:
   ```
   python upload_medicines.py \
     --base-url https://YOUR-NEW-URL.ngrok-free.app/ \
     --email YOUR_EMAIL \
     --password YOUR_PASSWORD
   ```
4. افتح تبويب الـ Store في التطبيق و اعمل refresh — هتلاقي المنتجات ظهرت.

## ملاحظات
- الرابط بتاع ngrok المجاني بيتغير كل مرة تقفل/تفتح ngrok، فلازم تمرّر الرابط الحالي.
- لو ظهر خطأ 401/403: التوكن أو بيانات اللوجين غلط. لو 422: السيرفر مستني حقول مختلفة — ابعتلي رد السيرفر وأظبط السكريبت.
